## 初めての方へ

### Voice Studioとは？

Voice Studioは、テキストから音声を生成したり、動画から文字起こしをしたりできるアプリです。

**できること:**
- 📖 テキストを読み上げ音声に変換
- 🎭 様々なキャラクター・感情で音声生成
- 🎬 動画ファイルから自動文字起こし
- 📝 字幕ファイル（SRT）の自動作成

### 基本操作

#### 画面の見方

```
┌─────────────────────────────────────────┐
│  タブ選択  │ VOICEVOX │ Coqui │ Whisper│
├─────────────────────────────────────────┤
│                                         │
│  【テキスト入力エリア】                │
│  ここに読み上げたい文章を入力します    │
│                                         │
├─────────────────────────────────────────┤
│  キャラクター選択 ▼                     │
│  感情選択 ▼                              │
│  話速調整 ━━●━━                         │
│  音高調整 ━━●━━                         │
├─────────────────────────────────────────┤
│         [音声生成ボタン]                │
└─────────────────────────────────────────┘
```

## VOICEVOX音声合成

### ステップ1: VOICEVOXの準備

Voice Studioを使う前に、VOICEVOXをインストールして起動しておく必要があります。

#### VOICEVOXのインストール（初回のみ）

1. [VOICEVOX公式サイト](https://voicevox.hiroshiba.jp/)にアクセス
2. 「ダウンロード」ボタンをクリック
3. ダウンロードしたファイルを実行してインストール
4. インストール完了後、VOICEVOXを起動

#### VOICEVOXの起動確認

VOICEVOXが正しく起動しているか確認してください:
- タスクバーにVOICEVOXのアイコンが表示されている
- VOICEVOXのウィンドウが開いている

### ステップ2: 音声生成

#### 基本的な手順

1. **タブ選択**: 「VOICEVOX」タブをクリック
2. **テキスト入力**: 読み上げたい文章を入力
   ```
   例: こんにちは。老後AIのたけ爺です。
   ```
3. **キャラクター選択**: お好みのキャラクターを選択
   - ずんだもん
   - 四国めたん
   - 春日部つむぎ
   - など多数
4. **感情選択**: キャラクターの感情を選択
   - ノーマル
   - 喜び
   - 怒り
   - 悲しみ
   - など
5. **音声生成**: 「音声生成」ボタンをクリック
6. **保存**: ファイル名と保存先を指定

#### 詳細設定

##### 話速調整
- **スライダー**: 左右に動かして調整
- **範囲**: 0.5倍速〜2.0倍速
- **推奨**: 
  - ゆっくり: 0.8倍速
  - 標準: 1.0倍速
  - 速い: 1.3倍速

##### 音高調整
- **スライダー**: 左右に動かして調整
- **範囲**: -12〜+12（半音単位）
- **推奨**: 
  - 低い声: -3〜-5
  - 標準: 0
  - 高い声: +3〜+5

### ステップ3: ファイル保存

#### 保存形式
- **WAV形式**: 無圧縮、高品質
- **MP3形式**: 圧縮、ファイルサイズ小

#### ファイル名の付け方（推奨）

```
例1: 2025-01-06_intro_zundamon.wav
  ├─ 日付
  ├─ 内容の説明
  └─ キャラクター名

例2: video01_narration_tsumugi.mp3
  ├─ プロジェクト名
  ├─ 用途
  └─ キャラクター名
```

## Coqui TTS音声合成

### ステップ1: モデル選択

Coqui TTSは多言語対応の音声合成エンジンです。

#### 利用可能な言語
- 日本語（Japanese）
- 英語（English）
- 中国語（Chinese）
- スペイン語（Spanish）
- その他多数

### ステップ2: 音声生成

#### 基本的な手順

1. **タブ選択**: 「Coqui TTS」タブをクリック
2. **テキスト入力**: 読み上げたい文章を入力
3. **モデル選択**: 言語に合ったモデルを選択
4. **話速調整**: お好みの速度に調整
5. **音声生成**: 「音声生成」ボタンをクリック
6. **保存**: ファイル名と保存先を指定

#### Coqui TTSの特徴

**利点:**
- 多言語対応
- VOICEVOXのインストール不要
- オフラインで動作

**欠点:**
- VOICEVOXより自然さは劣る
- 日本語の精度がVOICEVOXより低い

**使い分けの目安:**
- 日本語音声 → VOICEVOX推奨
- 英語音声 → Coqui TTS推奨
- 多言語ミックス → Coqui TTS

## Whisper文字起こし

### 概要

Whisper機能は、MP4動画ファイルから音声を抽出し、テキストに変換します。

### ステップ1: 動画ファイルの準備

#### 対応フォーマット
- MP4（推奨）
- AVI
- MOV
- MKV

#### ファイルサイズの目安
- 10分の動画: 約50-200MB
- 30分の動画: 約150-600MB

### ステップ2: モデルの選択

#### 3種類のモデル

| モデル | 速度 | 精度 | 推奨用途 | 処理時間（10分動画） |
|--------|------|------|----------|----------------------|
| **base** | ⚡⚡⚡ | ★★☆ | テスト・下書き | 2-3分 |
| **medium** | ⚡⚡ | ★★★ | **推奨・実用** | 5-7分 |
| **large-v3** | ⚡ | ★★★★ | 最高精度が必要 | 10-15分 |

#### 選び方のコツ

**初めての方:**
- まず**mediumモデル**を試してください
- 95%の精度で実用的です

**baseモデルが向いている場合:**
- とにかく速く結果が欲しい
- 大まかな内容確認でOK
- 後で手動修正する予定

**large-v3が向いている場合:**
- 専門用語が多い動画
- 複数人の会話
- 最高品質が必要

### ステップ3: 文字起こし実行

#### 基本的な手順

1. **タブ選択**: 「Whisper」タブをクリック
2. **ファイル選択**: 「動画ファイルを選択」ボタンをクリック
3. **動画選択**: MP4ファイルを選択
4. **モデル選択**: mediumを推奨
5. **実行**: 「文字起こし開始」ボタンをクリック
6. **待機**: 進行状況バーを確認
7. **完了**: SRTファイルが自動保存されます

#### 処理中の表示

```
処理中...
━━━━━━━●━━━━━━━ 45%
推定残り時間: 3分20秒
```

### ステップ4: 結果の確認

#### 生成されるファイル

**元の動画:**
```
C:\Users\YourName\Videos\sample.mp4
```

**生成されるSRTファイル:**
```
C:\Users\YourName\Videos\sample.srt
```

#### SRTファイルの中身（例）

```srt
1
00:00:00,000 --> 00:00:03,500
こんにちは。老後AIのたけ爺です。

2
00:00:03,500 --> 00:00:08,200
今日はWhisper機能の使い方を説明します。

3
00:00:08,200 --> 00:00:12,800
この機能を使えば、動画から自動で字幕が作れます。
```

### ステップ5: 字幕の活用

#### 動画編集ソフトでの使用

**DaVinci Resolve:**
1. 字幕トラックを追加
2. SRTファイルをインポート
3. タイムラインに配置

**Adobe Premiere Pro:**
1. ファイル → 読み込み → SRTファイル
2. 字幕トラックにドラッグ

**無料の動画編集ソフト:**
- Shotcut（無料）
- OpenShot（無料）

#### YouTubeへのアップロード

1. YouTube Studioを開く
2. 動画をアップロード
3. 「字幕」→「ファイルをアップロード」
4. SRTファイルを選択

## 設定とカスタマイズ

### 保存先の変更

#### デフォルトの保存先を設定

1. 「設定」メニューを開く
2. 「デフォルト保存先」を選択
3. お好みのフォルダを選択

## よくある質問

### Q1: VOICEVOXが見つからないと表示される

**A**: VOICEVOXを起動していますか？
1. VOICEVOXアプリを起動
2. Voice Studioを再起動
3. もう一度試してみてください

### Q2: 音声生成が遅い

**A**: 以下を確認してください:
- PCのメモリは十分ですか？（推奨8GB以上）
- 他のアプリを閉じてみてください
- テキストが長すぎる場合は、分割してみてください

### Q3: Whisper処理が途中で止まる

**A**: 以下の対策を試してください:
1. baseモデルに変更
2. 動画ファイルを短く分割
3. PCを再起動して再度実行

### Q4: 字幕のタイミングがずれる

**A**: Whisperの制限です:
- mediumモデルで95%の精度
- 完璧を目指す場合は手動修正が必要
- 動画編集ソフトでタイミング調整

### Q5: EXEファイルを移動したら動かなくなった

**A**: `_internal`フォルダと離れてしまったためです:
1. 元のフォルダに戻す
2. ショートカットを作成してデスクトップに配置
3. EXE本体は移動しない

### Q6: WindowsDefenderに警告が出る

**A**: 署名なしアプリの標準的な警告です:
1. 「詳細情報」をクリック
2. 「実行」をクリック
3. 安全なアプリです

### Q7: 日本語が文字化けする

**A**: ファイル名に特殊文字が含まれていませんか？
- ファイル名に使用可能:
  - 半角英数字
  - アンダースコア（_）
  - ハイフン（-）
- 避けるべき文字:
  - スペース
  - 全角文字（できれば）
  - 特殊記号（<>:"/\|?*）
